# Kalkulator Podróży

## Opis projektu
Aplikacja została stworzona, aby ułatwić szybkie szacowanie kosztów wyjazdu. Pozwala obliczyć wydatki na paliwo (z uwzględnieniem różnych cen za litr) oraz koszty noclegu. 

## Technologie
* **Język:** Java
* **Środowisko:** Android Studio

## Przykład działania
1. W głównym ekranie wpisz cel podróży (np. "Gdańsk").
2. Podaj liczbę kilometrów oraz koszt noclegu.
3. Wybierz rodzaj paliwa (Benzyna, Diesel, LPG) – aplikacja sama dobierze średnią cenę.
4. Kliknij **Oblicz**, aby przejść do podsumowania.
5. Na drugim ekranie zobaczysz wynik. Kliknij **Zatwierdź**, aby wrócić do formularza ze statusem "Zatwierdzono".
